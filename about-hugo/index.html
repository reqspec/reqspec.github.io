About

Юрий, сопровождаю процессы в Хоум Кредит Банке в рамках клиентского сервиса. Выявляю требования, пишу User Story. Использую Jira, Confluence, SQL, Tableau, Python.

  • Разработка технических заданий, проектирование моделей бизнес-процессов, формирование функциональных требований

  • Сбор, анализ, детализация бизнес-требований

  • Участие во внедрении и сопровождении программного продукта, сбор и анализ замечаний пользователей

  • Обучение пользователей в рамках произведенных изменений

  • Подготовка аналитических, справочно-информационных и презентационных материалов

  • Представление результатов работы стейкхолдерам

  • Тестирование разработок на предмет соответствия согласованным требованиям

  • Работа с продуктами Atlassian (Jira, Confluence, Trello)

  • Работа с SQL – Агрегатные функции(AVG,SUM,MAX,MIN,COUNT) – Стрококовые функции(CONCAT,INSERT,LOWER,UPPER) – Функции временного ряда(TO_DATE) – Условные выражения(CASE) – Использование операторов(BETWEEN,IS NULL,LIKE)

  • Tableau – Поддержка текущих дашбордов – Разработка новых согласно требованиям бизнеса

  • Python – Написание простых скриптов для оптимизации деятельности(Beautifulsoup,Requests,Selenium,Pandas)

  • Аудит деятельности подразделений в рамках процессов, для выявления болей клиентов и недоработок процессов

  • Ведение текущей отчетности отдела(MS Excel (сводные таблицы, ВПР, вложенные формулы и т.д.)

  • Работа с хранилищем данных(DWH) как пользователь – Формирование выгрузок и отчетности на основе полученных данных – Добавление новых данных в существующие таблицы, создание новых таблиц – Создание процедур, работа с планировщиком

  • Разработка user story в рамках бизнес требований стейкхолдеров

  • Разработка и реализация решений автоматизации бизнес-процессов
